# Проектная работа "Научиться учится"

### Одностраничный сайт, который рассказывает о различных техниках обучения. На этом сайте можно найти как видео, так и ссылки с полезной информацией.
### На странице можно имеется желательное к просмотру видео) Само видео находится на YouTube, но благодаря iframe API YouTube, они были добавлены на сайт. Также на сайте можно увидеть анимированные фигуры с импользованием Кейфрейма и CSS свойста animation. При написании кода, все стили были написаны по файловой структуре БЭМ nested.

# Что требуется для открытия кода

### Данная страница не имеет сбору с помощью WebPack, поэтому для открытия этого сайта у себя на компьютере достаточно лишь клонировать себе репозиторий "git clone https://github.com/Vinorval/how-to-learn.git".
